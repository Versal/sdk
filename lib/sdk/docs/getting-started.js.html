Getting started

This guide will help you generate your first gadget and find your way around Versal's SDK.

1) Install the Versal SDK

Versal's SDK is a command line tool installable via Node.js's package manager.

Prerequisites:

  • Node.js (and NPM)

  • During the beta period your Versal developer representative will give you access to the SDK repository (eventually it'll be installable by name via npm). For this you'll need a github account.

From the terminal run:

npm install -g git+ssh://git@github.com:Versal/sdk.git

To verify the command line tool is available run the versal command.

2) Generate your gadget

The SDK init subcommand will generate all the files needed to build a gadget.

To generate a basic gadget named hello-gadget run:

mkdir hello-gadget
cd hello-gadget
versal init

Read SDK Commands | init for more info about the init subcommand and Anatomy of a gadget for more info about building gadgets.

3) Preview your gadget

The SDK's preview will run a local server for previewing and manually testing your gadget:

versal preview

Read SDK Commands | preview for more info about the preview subcommand.

4) Publish your gadget

The SDK's publish subcommand will publish your gadget to the Versal platform for review:

versal publish

Read SDK Commands | publish for more info about the publish subcommand.
