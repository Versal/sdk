SDK Reference

Commands

init

Generates all the files needed to build a gadget in the current directory.

versal init

The default gadget has the following structure:

├── README.md
├── assets
│   └── icon.png
├── gadget.css
├── gadget.js
├── index.html
├── manifest.json
├── package.json
├── scripts
│   └── random.js
├── sdk
│   [...]
└── test
    ├── another_spec.js
    └── gadget_spec.js

preview

Runs a local server for previewing and manually testing your gadget:

versal preview

UI

The UI simulates the environment of the Course Player that your final gadget will run in. Radio buttons labeled "Teacher" and "Student" for triggering the events that move the gadget from the teacher's editable view to the student's interactive view.

Debugging

All events that the player sends and receives are logged to the browser console

publish

Publish yours gadget to the Versal platform for review:

versal publish
